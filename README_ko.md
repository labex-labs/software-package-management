소프트웨어 패키지 관리를 학습합니다. 이 모듈은 패키지 관리자인 APT 및 DNF, 패키지 검색, 소프트웨어 설치 및 제거, 시스템 업데이트 및 업그레이드, 리포지토리 관리를 다룹니다. 실습 랩과 실제 과제를 통해 이러한 필수 Linux 기술을 숙달하게 됩니다.
