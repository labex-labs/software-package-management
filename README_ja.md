ソフトウェアパッケージ管理について学習します。このモジュールでは、パッケージマネージャーである APT と DNF、パッケージの検索、ソフトウェアのインストールと削除、システムアップデートとアップグレード、リポジトリの管理について扱います。実践的なラボと実世界の課題を通じて、これらの必須の Linux スキルを習得します。
